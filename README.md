Cross-Border Data Transfer Compliance & Threat Monitoring System


This project simulates a secure cross-border healthcare data transfer system with integrated cybersecurity monitoring.

The system includes:

- API authentication (API key validation)
- Payload integrity verification (SHA-256 signature hashing)
- Compliance risk scoring engine
- Rule-based attack detection
- Machine learning anomaly detection
- Security logging
- Real-time monitoring dashboard

## Architecture

Hospital Node → Secure API → Authentication Layer → Integrity Check → Attack Detection → ML Anomaly Detection → Compliance Engine → Logging → Dashboard

## Security Features

1. API Key Authentication
2. SHA-256 Payload Signature Verification
3. Rate-Based Attack Detection
4. Isolation Forest ML Anomaly Detection
5. Compliance Risk Scoring
6. Structured Security Logging
7. Web-Based Monitoring Dashboard

## Technologies Used

- Python
- Flask
- Scikit-Learn
- Isolation Forest
- SHA-256 Hashing
- REST API Architecture

## How to Run

1. Activate environment
2. Run:
   python backend/api_server.py
3. Open:
   http://127.0.0.1:5001 (Dashboard)
